# ChatCensus
WoW 1.12 addon to census chat channels

Usage:
/ccensus CHANNELNAME to poll the current members of chosen chat channel. Custom channels require the channel name itself e.g. /ccensus world, whereas default channels only require the channel number e.g. /ccensus 1 for General.
